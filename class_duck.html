◆ getSignalScore()

template<typename WifiCapability = DuckWifiNone, typename RadioType = DuckLoRa>
SignalScore Duck< WifiCapability, RadioType >::getSignalScore ( )
inlineprotected

Calculate a signal score based on the current RSSI and SNR values. The signal score is a value between 0 and 1, where 1 is the best possible signal and 0 is the worst possible signal.

The formula used to calculate the signal score is:

signalScore = (normalizedRSSI + normalizedSNR) / 2

where:

normalizedRSSI = (RSSI - RSSI_MIN) / (RSSI_MAX - RSSI_MIN) normalizedSNR = (SNR - SNR_MIN) / (SNR_MAX - SNR_MIN)

RSSI_MIN = -131 dBm RSSI_MAX = -20 dBm SNR_MIN = -11.5 dB SNR_MAX = 11.5 dB

Returns
SignalScore struct containing rssi, snr, and signalScore values
